# EPICS-Grammar-Course-Redesign

# Requirements

1. VS code
2. Live Server

Clone the repo to your system. Open the folder in VS code. Install live server through the extentions tab. After installation navigate to the index.html file and right click to find the Open Live Server option (Or use Alt+L Alt+O, up to you IDC). This should open the website in your default browser, It might take a few seconds, that's normal (Ur browser sucks). Test as you want.
